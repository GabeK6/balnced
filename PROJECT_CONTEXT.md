Balnced is a Next.js budgeting web app.

Purpose:
Help users understand how much money they can safely spend between paychecks.

Core calculations:
- current balance
- expected paycheck
- upcoming bills
- safe to spend
- daily spending limit

Main files:
app/dashboard/page.tsx
lib/api/dashboard-data.ts
app/projection/page.tsx

Current feature being built:
Recurring payday system supporting salary and hourly pay.
